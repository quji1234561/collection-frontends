清除浮动

问题：给父元素设置高度无法适配浮动子元素的，不设置高度父元素高度又会变为零，空间被下方的元素挤占

需要清除浮动的情况：
1.父元素没高度；
2.子元素浮动；
3.影响下方布局

解决方法：清除浮动（清除浮动造成的影响）

清除浮动后，父元素会自动检测子元素的高度，产生相应高度，不影响下方的标准流

原理是闭合浮动，将浮动子元素的影响限制在父元素内

clear:属性值

left不允许左浮动（清除左浮动影响）

right不允许右浮动（清除右浮动影响）

both同时清除两侧影响（最常用）

清除浮动方法：

1.额外标签法（隔墙法）

在最后一个浮动的子元素后加一个样式clear:both的空块级元素或br

优缺点：方便，但结构化差

2.父元素加overflow

属性值有hidden auto scroll

用hidden或auto

优缺点：简洁，但无法显示溢出的部分

3.父元素加after伪元素

看源码注释：

.clearfix:after{
content:"";
display:block;
height:0;
clear:both;
visibility:hidden;
}
ie6、7专有：
.clearfix{
*zoom:1;
}

优缺点：结构简单，但要照顾低版本的浏览器

4.父元素加双伪元素

.clearfix:before,
.clearfix:after{
content:"";
display:table;
}
.clearfix:after{
clear:both;
}
.clearfix{
*zoom:1;
}

优缺点：结构简单，但要照顾低版本的浏览器
